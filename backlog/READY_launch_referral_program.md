# Launch Referral Program
**Owner:** Growth Team / Backend Engineer  
**Objective:** Generate 20% of new customers organically via referrals, reducing blended CAC.

## Steps:
1. Review experiment plan: `/growth/experiments/referral-program/plan.md`
2. Build referral system:
   - Unique referral code generation
   - Attribution tracking (UTM parameters)
   - Credit issuance system ($25 × 2)
   - Fraud detection (duplicate emails, IPs)
3. Create referral UI:
   - "Refer Friends" button in app
   - Referral link sharing (email, SMS, social)
   - Referral dashboard (track referrals, credits)
4. Set up email/SMS automation:
   - Referral invite templates
   - Credit notification emails
5. Soft launch (Week 1):
   - Email to top 10% customers
   - Monitor for abuse/fraud
6. Full rollout (Week 2+):
   - In-app prompts
   - Marketing campaign
7. Monitor weekly:
   - Referral signup rate
   - Blended CAC
   - Fraud/abuse detection

## Dependencies:
- Payment/credit system
- Email/SMS automation platform
- Fraud detection logic

## KPI:
- 20% of new customers from referrals | **30-day signal:** Referral signups > 5% of total within first week

## Done when:
- Referral program live and functional
- 20% of customers coming from referrals
- Blended CAC decreases by 15% ($150 → $127.50)
- Fraud rate < 5%

**Impact:** High | **Confidence:** Medium | **Effort:** High (2-3 weeks)
