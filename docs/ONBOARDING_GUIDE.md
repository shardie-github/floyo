## Welcome to Floyo

Welcome to Floyo! We're excited to help you optimize your workflow and discover tools that fit your coding patterns.

### What is Floyo?

Floyo automatically tracks how you use files and tools, then provides AI-powered suggestions for integrations and optimizations. All while keeping your code completely private.

### What You'll Learn

This guide will walk you through:
- Setting up your account
- Understanding your dashboard
- Connecting integrations
- Managing privacy settings
- Getting the most out of Floyo

## Quick Start Guide

### Step 1: Create Your Account

1. Visit [floyo.dev](https://floyo.dev)
2. Click "Sign Up"
3. Enter your email and password
4. Verify your email address
5. Complete your profile

**Time:** ~2 minutes

### Step 2: Install the Extension (Optional)

1. Download the Floyo browser extension
2. Install in your browser
3. Log in with your account
4. Grant necessary permissions

**Time:** ~1 minute

### Step 3: Explore Your Dashboard

1. Navigate to Dashboard
2. Review your initial patterns (if any)
3. Check out the Insights panel
4. Explore available integrations

**Time:** ~5 minutes

## First Steps

### Complete Your Profile

1. Go to **Profile** (click your avatar)
2. Add your name and profile picture
3. Set your preferences
4. Review privacy settings

**Why it matters:** A complete profile helps personalize your experience.

1. Go to **Settings** → **Privacy**
2. Review data retention settings
3. Configure monitoring preferences
4. Set up app allowlists

**Why it matters:** You control what data is tracked and how long it's stored.

### Connect Your First Integration

1. Go to **Integrations**
2. Browse available integrations
3. Click "Connect" on an integration
4. Follow the OAuth flow
5. Verify connection status

**Why it matters:** Integrations unlock powerful automation features.

## Privacy & Security

### What Data is Tracked?

Floyo tracks:
- ✅ File paths and names
- ✅ File usage patterns
- ✅ Tool usage
- ✅ Timestamps
- ❌ File content (never)
- ❌ Sensitive data (never)

### Privacy Controls

**Data Retention:**
- Set how long data is stored (7-90 days)
- Automatic deletion after retention period
- Manual deletion anytime

**Monitoring:**
- Enable/disable monitoring per app
- Configure signal toggles
- Set sampling rates

**Export & Delete:**
- Export all your data (GDPR)
- Delete account and all data
- Request data deletion

### Security Features

- **Encryption:** All data encrypted in transit and at rest
- **Authentication:** Secure login with optional 2FA
- **Audit Logs:** Track all account changes
- **RLS:** Row-level security in database

## Troubleshooting

### Dashboard Not Loading

**Symptoms:** Dashboard shows loading spinner indefinitely

**Solutions:**
1. Refresh the page
2. Check internet connection
3. Clear browser cache
4. Try incognito mode
5. Contact support if issue persists

### Patterns Not Appearing

**Symptoms:** No patterns shown in dashboard

**Solutions:**
1. Ensure monitoring is enabled
2. Check privacy settings
3. Verify app allowlist
4. Wait 24 hours for initial data
5. Check browser extension is installed

### Integration Connection Fails

**Symptoms:** Can't connect to integration

**Solutions:**
1. Check API credentials
2. Verify OAuth permissions
3. Check integration status page
4. Try disconnecting and reconnecting
5. Contact support with error details

### Data Not Syncing

**Symptoms:** Recent activity not appearing

**Solutions:**
1. Check internet connection
2. Verify extension is running
3. Check privacy settings
4. Review app allowlist
5. Restart browser extension

## FAQ

### General Questions

**Q: Is my code content tracked?**  
A: No, Floyo only tracks file paths and usage patterns, never file content.

**Q: How much does Floyo cost?**  
A: Floyo offers a free tier. Pro is $29/month. Enterprise pricing is custom.

**Q: Can I use Floyo offline?**  
A: Floyo works offline and syncs when you're back online.

**Q: Is Floyo GDPR compliant?**  
A: Yes, Floyo is GDPR-compliant with full data export and deletion capabilities.

### Technical Questions

**Q: What browsers are supported?**  
A: Chrome, Firefox, Safari, and Edge (latest versions).

**Q: Do I need to install anything?**  
A: The browser extension is optional but recommended for best experience.

**Q: How is data stored?**  
A: Data is encrypted and stored in secure databases with row-level security.

**Q: Can I self-host Floyo?**  
A: Self-hosting is available for Enterprise customers.

### Privacy Questions

**Q: Who can see my data?**  
A: Only you can see your data. We never share data with third parties.

**Q: Can I delete my data?**  
A: Yes, you can delete your account and all data anytime.

**Q: How long is data stored?**  
A: You control data retention (7-90 days) or unlimited for Pro/Enterprise.

**Q: Is data encrypted?**  
A: Yes, all data is encrypted in transit and at rest.
