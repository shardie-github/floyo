## Shopify App Store Requirements

### Compliance Checklist
- ✅ **Functionality:** App works as described (no crashes, broken features)
- ✅ **Privacy:** Privacy policy linked, data collection disclosed
- ✅ **Shopify API:** Compliant with Shopify API usage guidelines
- ✅ **Support:** Support URL provided, contact information available

### Review Guidelines Responses

**Q: Does your app require internet access?**
A: Core functionality works offline (file monitoring, pattern detection). Internet access is required for Shopify API integrations when workflows are executed.

**Q: Does your app collect user data?**
A: Minimal data collection, opt-in only (usage analytics, crash reports). All file processing happens locally—no user files or Shopify data are uploaded or stored in the cloud.

**Q: Does your app require special permissions?**
A: File system access permission (required for file monitoring). Shopify API access (OAuth, scopes: read_products, write_products, read_orders, write_orders).
