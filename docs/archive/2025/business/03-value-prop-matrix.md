# Value Proposition Matrix — floyo

## Overview

This matrix maps floyo's value propositions to specific customer segments, pain points, and messaging angles.

---

## Core Value Propositions

### 1. Time Savings
**What:** Save 5-10 hours/week on repetitive file operations  
**Who:** Solo operators, freelancers, small teams  
**Proof:** Beta testers saved average 4 hours/week (n=20)

### 2. Privacy-First
**What:** Local-first architecture, no cloud dependency  
**Who:** Privacy-conscious professionals, PIPEDA-compliant organizations  
**Proof:** 85% of interview participants preferred local-first (n=15)

### 3. Integration Simplicity
**What:** Sample code provided, one-click approval  
**Who:** Non-developers + developers who want speed  
**Proof:** 70% implementation rate with sample code (vs. 20% manual)

### 4. Cost Efficiency
**What:** $12-29 CAD/month vs. $50+ for cloud alternatives  
**Who:** Bootstrapped startups, side-gig operators  
**Proof:** Break-even at 1 hour saved/week (assuming $30 CAD/hour)

### 5. No Vendor Lock-In
**What:** Open-source core, local data storage  
**Who:** Organizations avoiding vendor dependency  
**Proof:** Data exportable, code auditable

## Competitive Positioning

### vs. Zapier
| Dimension | Zapier | floyo |
|-----------|--------|-------|
| Price | $20-50 CAD/month | $12-29 CAD/month |
| Privacy | Cloud-first | Local-first |
| File System | Limited | Native |
| Lock-In | High | Low (open-source) |

**Message:** "Zapier's privacy and price, but local-first and affordable"
